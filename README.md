# Proyecto-Mecatr-nica-Grupo-1
Este repositorio reúne todos los archivos necesarios para replicar el robot auto-balanceado del proyecto: modelos 3D, esquemáticos electrónicos, código del controlador PID, manual de armado y lista de materiales. Permite reproducir el diseño mecánico, eléctrico y funcional del prototipo final.
